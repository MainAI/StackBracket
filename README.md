## StackBracket
*Check the balance of brackets:*  
*Bracket balance means that each opening character has a corresponding closing character, and pairs of brackets nest correctly.*

expects sample sequences:  
* [([])((([[[]]])))]{()}
* {{[(])]}}
* {test{text[(my)be]other}}

result print:
* "Сбалансированно"
* "Несбалансированно"
